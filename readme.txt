=== Simple Show Card ===
Contributors: Kevin Lee

* Generated with shortcode
* Accept a parameter being the card id
* Accept a language parameter to either display in fr or en

= How it works =
* Need to add short code
* [simpleshowcard card_id='5f342a3992ec22115033b2fb' card_lang='fr' card_imgsize='200']

== Installation ==

1. Upload `simple-show-card` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
